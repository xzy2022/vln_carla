**结论（UE5）**  
有可用接口，能完成你说的两件事：坐标认知 + 指定点生成测试物体。

1. 坐标原点与轴方向  
- `carla.Location.x/y/z` 明确是“到 origin 的距离”，可把 `(0,0,0)` 当世界原点参考点：`Docs_Carla_UE5/python_api.md:1308`  
- 坐标系是 UE 的 `Z-up left-handed`：`Docs_Carla_UE5/python_api.md:1651`  
- 可用 `world.debug.draw_arrow/draw_point/draw_string` 在场景里画坐标轴：`Docs_Carla_UE5/python_api.md:711`、`Docs_Carla_UE5/python_api.md:715`  
- 可用 `Transform/Rotation.get_forward_vector/get_right_vector/get_up_vector` 获取方向向量：`Docs_Carla_UE5/python_api.md:1671`、`Docs_Carla_UE5/python_api.md:2231`  
- `+X forward, +Y right, +Z up` 是基于 UE 左手系的推断（文档该段未逐字写出 X/Y 语义）。

2. 在特殊点生成物体  
- 直接用 `world.spawn_actor()` 或 `world.try_spawn_actor()`（失败返回 `None`）：`Docs_Carla_UE5/python_api.md:3256`、`Docs_Carla_UE5/python_api.md:3271`  
- 先用 `world.get_blueprint_library()` 选蓝图：`Docs_Carla_UE5/python_api.md:3301`  
- 道具蓝图可用 `static.prop.*`（如 `static.prop.barrel`）：`Docs_Carla_UE5/catalogue_props.md:127`  
- 交通灯/停车标志这类 OpenDRIVE 自动对象不能通过蓝图新生成：`Docs_Carla_UE5/core_actors.md:187`

3. 选“特殊点”的辅助接口  
- `map.get_spawn_points()`、`map.get_waypoint()`、`map.generate_waypoints()`、`map.get_topology()`：`Docs_Carla_UE5/python_api.md:1400`、`Docs_Carla_UE5/python_api.md:1406`、`Docs_Carla_UE5/python_api.md:1360`、`Docs_Carla_UE5/python_api.md:1403`  
- `world.get_level_bbs()` 可拿到场景包围盒，便于找边界/空旷区域：`Docs_Carla_UE5/python_api.md:3309`  
- `world.ground_projection()` 可把测试点投影到地面，减少悬空或穿模：`Docs_Carla_UE5/python_api.md:3228`

- 已有 `get_map/get_spawn_points/spawn_actor/get_spectator` 逻辑：`src/infrastructure/carla/carla_env_adapter.py:224`、`src/infrastructure/carla/carla_env_adapter.py:229`、`src/infrastructure/carla/carla_env_adapter.py:255`
