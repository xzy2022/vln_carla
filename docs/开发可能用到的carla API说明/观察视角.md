UE5 下你在“仿真可视化窗口”看到的视角，本质是 `spectator`（观察者相机）。

**可视化窗口里手动移动**
- 启动后出现的是 spectator 视角：`Docs_Carla_UE5/start_quickstart.md:94`
- 文档给的飞行控制是 `Q/E/W/A/S/D + 鼠标`：`Docs_Carla_UE5/tuto_G_getting_started.md:102`
- 按键定义（上下前后左右）在这里：`Docs_Carla_UE5/tuto_G_getting_started.md:104`
- 该窗口可被 `-RenderOffScreen` 关闭：`Docs_Carla_UE5/tuto_G_getting_started.md:98`

**编程接口（设置/移动视角）**
- 获取 spectator：`world.get_spectator()`，它就是控制模拟器窗口视角的 actor：`Docs_Carla_UE5/python_api.md:3331`
- 读当前位姿：`spectator.get_transform()`：`Docs_Carla_UE5/tuto_G_getting_started.md:123`
- 设置位姿：`spectator.set_transform(carla.Transform(...))`：`Docs_Carla_UE5/tuto_G_getting_started.md:129`
- `set_transform` 语义是“传送到指定位置+朝向”：`Docs_Carla_UE5/python_api.md:140`
- 文档示例里也直接把 spectator 放到车辆位置：`Docs_Carla_UE5/python_api.md:4043`
- 若要持续跟随移动目标，通常每 tick 更新一次（`tick/wait_for_tick/on_tick`）：`Docs_Carla_UE5/python_api.md:3264`、`Docs_Carla_UE5/python_api.md:3284`、`Docs_Carla_UE5/python_api.md:3238`

**回放模式相关视角接口**
- `client.replay_file(..., follow_id, ...)` 可让回放相机跟随某 actor：`Docs_Carla_UE5/python_api.md:494`
- `follow_id=0` 时相机跟随关闭：`Docs_Carla_UE5/python_api.md:500`
- 可设置是否忽略录制的 spectator 运动：`set_replayer_ignore_spectator`：`Docs_Carla_UE5/python_api.md:583`
