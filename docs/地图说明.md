关于后缀和特殊图：

*_Opt（例如 Town01_Opt、Town10HD_Opt）：分层地图（Layered map），布局与对应的非 _Opt 版本相同，但可通过 load_map_layer / unload_map_layer 切换建筑、植被、停放车辆等图层；最小布局（道路、人行道、信号灯等）始终保留。见 core_map.md 的 “Layered maps” 章节

Town10HD / Town10HD_Opt：你仓库文档里只有“加载示例”，没有具体地图特性描述。文档提到可以 client.load_world("Town10HD_Opt")，但没说明与 Town10 的差异。见 adv_synchrony_timestep.md

AnnotationColorLandscape：在当前文档中未找到说明（Docs_Carla 与 PythonAPI_Carla 都没有描述）。它应该是用于某类标注或可视化用途的内部地图，但这属于推测，文档未给出官方说明。
