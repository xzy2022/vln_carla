## 地图版本与分类

本文按引擎版本区分 CARLA 常见地图，并给出简要用途说明。实际可用地图以 `client.get_available_maps()` 的返回结果为准。

### UE4 体系地图（CARLA 0.9.x 常见）

- `Town01` / `Town01_Opt`：小镇道路，含河流与多桥，T 字路口较多，适合基础路口导航测试。
- `Town02` / `Town02_Opt`：小镇与公园混合场景，含商业区、住宅区、加油站，适合中低复杂度交通流测试。
- `Town03` / `Town03_Opt`：较大城市路网，含复杂路口与环岛，适合城市决策与多冲突场景测试。
- `Town04` / `Town04_Opt`：山地/坡道场景较多，含立交和起伏道路，适合纵向控制与坡道路测。
- `Town05` / `Town05_Opt`：多车道城市主干道较多，路口尺度更大，适合高车流密度测试。
- `Town06` / `Town06_Opt`：高速与快速路特征明显，含匝道与并线场景，适合中高速驾驶策略测试。
- `Town07` / `Town07_Opt`：乡村风格道路，车道更窄、信号控制更少，适合低结构化道路测试。
- `Town10HD` / `Town10HD_Opt`：高精度城区地图（HD），适合更细粒度感知与城市场景评估。

- `Mine_01`: 矿区非结构地图。
- `Town10HD_Opt`: 城区地图，默认地图。

### 关于 `_Opt` 后缀（分层地图）

- `_Opt` 版本与同名非 `_Opt` 地图在道路拓扑上保持一致。
- `_Opt` 支持按层加载/卸载场景元素（如建筑、植被、停放车辆等），便于性能优化。
- 最小道路骨架（道路、人行道、交通信号等）会保留。

## 查看当前环境可用地图

```python
import carla  # type: ignore

client = carla.Client("localhost", 2000)
print("可用地图列表:", client.get_available_maps())
```
